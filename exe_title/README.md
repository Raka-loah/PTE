# 将网易云音乐歌名输出到OBS

![](https://img.shields.io/github/stars/Raka-loah/PTE.svg)
[![](https://img.shields.io/github/license/Raka-loah/PTE.svg)](https://github.com/Raka-loah/PTE/blob/master/LICENSE)

**前置条件**
------

.net Framework 4.7.2

**使用方法**
------

1. 把代码Clone到本地；
2. 编译；
3. 运行；
4. 打开网易云音乐放歌；
5. 点击主界面的启动按钮；
6. 把music_title.txt添加到OBS的文本源。
